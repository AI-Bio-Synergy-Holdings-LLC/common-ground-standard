# Public Exposure Register

Status: Public exposure-control register
Date: June 21, 2026

## Purpose

This register records which project names, marks, and future-facing terms are visible in the public Common Ground Standard repository during the public-review phase.

The goal is to keep the public review clear, narrow, and critique-ready without accidentally turning protected names, internal architecture, future tools, or possible consortium structures into public claims.

## Current Rule

Use "Common Ground Standard" as the public-facing subject of this repository and website.

Use AI-Bio Synergy Holdings LLC only for stewardship, security, license, and funding-boundary context.

Do not use protected or future-facing names in narrative, outreach, roadmap, or review-brief language unless a later founder/legal review explicitly approves that use.

## Approved Public Uses

These uses are approved during the current public-review phase:

- review brief and website: Common Ground Standard;
- origin and stewardship: AI-Bio Synergy Holdings LLC;
- draft names: Campus Common Ground Compact and AI Dividend / Benefit Clause;
- legal and trademark notices: controlled names and marks, only as protective reservations;
- security policy: AI-Bio Synergy Holdings LLC as repository maintainer.

## Uses To Avoid For Now

Avoid using protected or future-facing names to describe:

- the public-facing project;
- active partnerships;
- operational networks;
- active products;
- active fiscal structures;
- active certification programs;
- active pilots;
- outreach claims;
- foundation or agency asks.

## Current Cleanup Decisions

The first public ambiguity sweep removed Pivot from front-door review language and replaced it with Common Ground Standard.

The roadmap should describe future capabilities generically unless a name is already part of the public review packet. For example, "evidence tooling" is safer than naming a future tool.

The public notice should avoid repeating protected internal names where a cross-reference to the license and trademark notice is enough.

## Review Triggers

Review this register before:

- adding new public pages;
- adding public sponsor or funding links;
- announcing a consortium or partnership;
- naming future tools or networks;
- adding marks to the website navigation;
- publishing outreach messages;
- creating certification or conformance language.
